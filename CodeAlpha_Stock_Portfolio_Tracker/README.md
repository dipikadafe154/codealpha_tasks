# Stock Portfolio Tracker 📈

## Overview

Stock Portfolio Tracker is a simple Python application that allows users to track their stock investments. Users can select stocks, enter the quantity of shares owned, and view the total investment value of their portfolio.

This project helps beginners understand Python dictionaries, loops, conditional statements, user input handling, and basic financial calculations.

## Features

* 📊 View available stocks and prices
* ➕ Add multiple stocks to a portfolio
* 💰 Calculate investment value for each stock
* 📈 Calculate total portfolio value
* ✅ Input validation
* ⚡ Beginner-friendly Python project

## Technologies Used

* Python 3

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Open Terminal.
5. Run the following command:

```bash
python stock_tracker.py
```

## How It Works

1. View the available stocks.
2. Enter a stock symbol.
3. Enter the quantity of shares.
4. Add multiple stocks if desired.
5. Type **done** to finish.
6. View the portfolio summary and total investment value.
