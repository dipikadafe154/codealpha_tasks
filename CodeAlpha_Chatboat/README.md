# Smart ChatBot 🤖

## Overview

Smart ChatBot is a simple Python-based chatbot that can interact with users through text commands. It responds to greetings, provides the current date and time, answers basic questions, and helps users through a command menu.

This project is designed for beginners who want to learn Python programming, loops, conditional statements, and user interaction.

## Features

* 👋 Responds to greetings
* 😊 Interactive conversation
* 📅 Shows current date
* ⏰ Shows current time
* ❓ Help menu with available commands
* 🚪 Exit command to end the chat
* ⚡ Simple and beginner-friendly code

## Technologies Used

* Python 3
* Datetime Module

## How to Run

1. Install Python 3 on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Open Terminal.
5. Run the following command:

```bash
python chatbot.py
```

## Available Commands

* hello
* hi
* hey
* how are you
* i am fine
* good
* what is your name
* who created you
* date
* time
* help
* bye
